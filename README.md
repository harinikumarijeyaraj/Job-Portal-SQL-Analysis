job Portal Analysis

📌 Overview

This project is a comprehensive SQL analysis of a job portal database, designed to demonstrate database design, data insertion, and analytical querying. It is implemented in both MySQL and Snowflake, making it a versatile reference for different SQL environments.

The project provides realistic sample data for jobs, companies, and applicants, and showcases how to extract meaningful insights from a job portal database. It is ideal for learning SQL, database analytics, and building portfolio-ready projects.

💻 Features
Design of a relational database schema for a job portal
SQL scripts for table creation and sample data insertion
Analytical queries to explore:
Recent job postings
Top hiring companies
Candidate applications per job and per company
Job category trends
Implemented in both MySQL and Snowflake SQL

🛠 Technology
SQL: MySQL, Snowflake SQL
Database Design: Normalized tables with relationships
Use Case: Job portal analytics and data insights

🚀 Getting Started
Clone the repository:

For MySQL:

For Snowflake:

Run snowflake/create_tables.sql
Run snowflake/insert_data.sql
Run snowflake/queries.sql to explore insights
